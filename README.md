# react-mobx
基于 [react-boilerplate-thunk](https://github.com/NoahZhang/react-boilerplate-thunk "NoahZhang")  
去掉了redux，用mobx来管理
  
执行 `npm install` 安装依赖包  
安装完之后 直接执行 `npm run start`  
浏览器打开 [localhost:8080](http://localhost:8080) 就能看到页面了
